# margin : auto

기능 : 해당 부분에 남은 margin을 모두 사용함.

용도 : margin을 특정 수치가 아닌 비율로서 모두 사용하고자 할 때, 사용하며, margin auto 사용 시 중앙정렬이 된다.(왼쪽과 오른쪽을 같은 비율로 모두 사용하므로)

예시 :   right-marign을 준 요소로 인하여 left 요소들이 우측으로 정렬되었다.
