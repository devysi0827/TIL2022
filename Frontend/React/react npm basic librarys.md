# react basic package

모두 필수는 아니지만 자주 사용하는 기능을 담은 package들입니다.

제가 다니는 직장 기준 필요한 요소들도 들어가있습니다.



### axios

```
npm install axios
```

### react-router

```
npm install react-router-dom@6
```

### styled-components

```
npm install --save styled-components
```

### typescript

```
npm install -g typescript
```

### mobx

```
npm install --save mobx
```
